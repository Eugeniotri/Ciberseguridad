Resumen claro de la UF1

Aquí tienes lo más importante del tema explicado de forma sencilla y pensada para examen.

1. DRP (Disaster Recovery Plan)

Es el plan de recuperación ante desastres. Sirve para restaurar los sistemas tras incidentes graves como incendios, apagones, inundaciones o ciberataques.

  • Inventario de hardware
  • Versiones de software
  • Configuración de red
  • Backups y recuperación
  • Documentación actualizada

2. SAI / UPS

El SAI proporciona energía temporal cuando se corta la luz. Evita apagados bruscos y da tiempo para apagar sistemas correctamente o arrancar el generador.

  • Offline: cambia a batería cuando falla la red
  • Inline: solución intermedia
  • Online: siempre alimenta desde batería

3. Mantenimiento del SAI

Las baterías se degradan con el tiempo, por eso hay que revisarlas periódicamente.

  • Pruebas periódicas
  • Reemplazo de baterías
  • Monitorización del estado
  • Apagado automático de servidores si hace falta

4. Seguridad física del CPD

Solo el personal autorizado debe poder acceder al CPD. Se protege con medidas físicas y de vigilancia.

  • Tarjetas de acceso
  • Biometría
  • Cámaras CCTV
  • Alarmas y vigilancia

5. Condiciones ambientales

Un CPD necesita temperatura y humedad controladas para evitar daños en los equipos.

  • Temperatura: 18°C – 24°C
  • Humedad: 40% – 60%
  • Filtros HEPA para el polvo
  • Pasillos fríos y calientes

6. Seguridad lógica

Protege el acceso a los sistemas informáticos. Aquí entran las ACL, contraseñas y la autenticación multifactor.

  • ACL: controla quién accede a qué
  • Menor privilegio: solo lo necesario
  • Denegación por defecto: todo bloqueado salvo lo permitido
  • MFA: más de un factor para autenticar

Esquema visual de memorización

Bloques rápidos para repasar antes del examen.

DRP

  • Plan para recuperar sistemas tras un desastre
  • Incluye hardware, software, red y backups
  • Idea clave: un DRP probado reduce la recuperación de días a horas

Tipos de SAI

  • Offline: barato, cambia a batería al fallar la red
  • Inline: intermedio, mejora la estabilidad
  • Online: máxima protección, sin interrupciones

Mantenimiento del SAI

  • Las baterías se degradan
  • Hay que hacer pruebas periódicas
  • Se deben reemplazar cuando toca
  • La monitorización permite detectar fallos

ACL

  • Access Control List
  • Controla acceso a recursos
  • Menor privilegio: solo acceso necesario
  • Denegación por defecto: todo bloqueado salvo lo permitido

MFA

  • Multi Factor Authentication
  • Combina varios factores
  • Algo que sabes → contraseña
  • Algo que tienes → móvil o token
  • Algo que eres → biometría

Biometría

  • Huella dactilar
  • Retina
  • Reconocimiento facial
  • Alta seguridad y evita suplantación

Condiciones del CPD

  • Temperatura: 18°C – 24°C
  • Humedad: 40% – 60%
  • Humedad baja → electricidad estática
  • Humedad alta → corrosión
  • Polvo → filtros HEPA

Seguridad física

  • Control de acceso
  • Cámaras de seguridad
  • Alarmas y vigilancia
  • Evita accesos no autorizados y robos físicos
Lo que más suele preguntar el profesor: tipos de SAI, DRP, ACL y menor privilegio, MFA, temperatura y humedad del CPD, biometría y mantenimiento del SAI.
